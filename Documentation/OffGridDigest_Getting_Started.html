Off-Grid Digest — Getting Started

One-time setup guide for forwarding unread iMessage/SMS and missed calls to your ZOLEO inbox.

1) Install and open the app

Open Off Grid Digest.app. A menu bar icon will appear.

The Swift app installs and controls the Go helper that does the forwarding work.

2) Fill in the basics

  • Toggle Enabled.
  • Set your Off-grid window.
  • Enter your Forwarding Email, usually your ZOLEO address.
  • Enter your Sender Email, the account Apple Mail should send from.
  • Optionally enter your ZOLEO Number for AI response handling.

3) Start the Go helper

From the app menu, click Reload Go Helper.

This installs and starts the LaunchAgent:

com.ampedig.off-grid-digest.helper

The helper runs in watch mode and checks every 60 seconds.

4) Grant permissions

macOS requires Full Disk Access for the installed Go helper:

$HOME/Library/Containers/com.ampedig.Off-Grid-Digest/Data/Library/Application Support/MsgForward/offgrid-digest

System Settings → Privacy & Security → Full Disk Access

You may also see an Apple Events / Automation prompt the first time Apple Mail sends a digest.

5) Verify

launchctl print gui/$(id -u)/com.ampedig.off-grid-digest.helper
tail -f "$HOME/Library/Containers/com.ampedig.Off-Grid-Digest/Data/Library/Application Support/MsgForward/OffGridDigest.log"

Send yourself a test iMessage. During the active window, the helper should detect unread 1-to-1 messages and send a digest email.

Troubleshooting

  • No emails? Re-check Full Disk Access for the installed Go helper binary.
  • Still nothing? Click Reload Go Helper, then tail OffGridDigest.log.
  • Apple Mail must be configured with the sender account.
  • Call history is best-effort because the database path and format can vary by macOS version.

Uninstall

launchctl bootout gui/$(id -u)/com.ampedig.off-grid-digest.helper 2>/dev/null || true
rm -f "$HOME/Library/LaunchAgents/com.ampedig.off-grid-digest.helper.plist"
rm -rf "$HOME/Library/Containers/com.ampedig.Off-Grid-Digest/Data/Library/Application Support/MsgForward"

Off‑Grid Digest — Getting Started

One‑time setup guide for forwarding iMessage/SMS and missed calls to your ZOLEO inbox.

1) Install the app

Drag Off Grid Digest.app to Applications and open it. A menu bar icon will appear.

2) Fill in the basics

  • Toggle Enabled.
  • Set your Off‑grid window (start/end).
  • Enter your forwardingEmail (your ZOLEO address).

3) Grant permissions (once)

macOS requires Full Disk Access for the tools the helper uses:

  1. System Settings → Privacy & Security → Full Disk Access
  2. Add /usr/bin/osascript and /usr/bin/sqlite3

You may also see an Apple Events prompt the first time email is sent.

4) Install & start the helper

From the app menu:

  • Click Install Helper (or launch once to auto‑install).
  • Click Reload LaunchAgent.

5) Verify

launchctl list | grep forward-messages
tail -f ~/Library/Logs/ForwardMessages.log

Send yourself a test iMessage; you should receive a digest email (during the active window).

Troubleshooting

  • No emails? Re‑check Full Disk Access for the two binaries above.
  • Still nothing? Click Reload LaunchAgent, then see the log at ~/Library/Logs/ForwardMessages.log.
  • Call history requires a readable DB at ~/Library/Application Support/CallHistoryDB/CallHistory.storedata (varies by macOS).

Uninstall

launchctl bootout gui/$(id -u) ~/Library/LaunchAgents/com.yourco.forward-messages.plist
rm -f ~/Library/LaunchAgents/com.yourco.forward-messages.plist
rm -f ~/Library/Scripts/ForwardMessagesToEmailWhileOffGrid.scpt
rm -rf ~/Library/Application\ Support/MsgForward
rm -f  ~/Library/Logs/ForwardMessages.log
